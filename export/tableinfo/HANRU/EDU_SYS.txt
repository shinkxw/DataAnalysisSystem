  ID            配置ID                  int            主键
  VALUE         配置值                  nvarchar(200)  非主

  Depth                                 int            非主
  PID                                   int            非主
  MenuDisp                              int            非主
  MenuUserType                          int            非主
  Link                                  nvarchar(100)  非主
  OrderIndex                            int            非主

  PICTYPE       图片类型                int            非主
  PIC1          图片一                  nvarchar(300)  非主
  PIC2          图片二                  nvarchar(300)  非主
