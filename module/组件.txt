每个组件只能完成较单一的工作
analyzer   分析器      将一种格式的字符串转换为元数据 需实现analyze方法
    BaseAnalyzer       分析器父类
    DafAnalyzer        daf格式
    SqlSSAnalyzer      sql sever脚本格式
    SqlPDAnalyzer      power designer脚本格式
    STAnalyzer         业务表数据

builder    生成器      将元数据装换为一种格式的字符串 需实现build方法
    BaseBuilder        生成器父类
    DafBuilder         daf格式
    SqlBuilder         sql脚本格式
    ModelBuilder       模型验证
    ViewBuilder        视图生成
    TemplateBuilder    模板生成

file       文件处理    文件读写相关
    DirManager         目录管理器
    FileLoader         文件读取器
    FolderLoader       文件夹读取器
    FileWriter         文件写入器
    FolderWriter       文件夹写入器
    AreaManager        元数据域管理器

helper      辅助工具   便于辅助开发及代码复用
    ArrReader     字符串数组读取器
    Indent        用于管理字符串的缩进
    Log           记录并管理日志

interface   系统界面   用户交互相关
    BaseInterface      命令行界面父类
    MainInterface      命令行主界面
    WorkSpaceInterface 工作空间界面
