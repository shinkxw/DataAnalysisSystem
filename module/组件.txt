每个组件只能完成较单一的工作
analyzer   分析器      将一种格式的数据转换为元数据
    DafAnalyzer        daf格式
    SqlSSAnalyzer      sql sever脚本格式
    STAnalyzer         业务表数据
    DBAnalyzer         数据库数据

builder    生成器      将元数据装换为一种格式的字符串
    Builder            泛用生成器
    DafBuilder         daf格式
    SqlBuilder         sql脚本格式
    ModelBuilder       模型验证
    ViewBuilder        视图生成
    TemplateBuilder    模板生成
    TestdataBuilder    测试数据脚本生成

file       文件处理    文件读写相关
    DirManager         目录管理器
    FileLoader         文件读取器
    FolderLoader       文件夹读取器
    FileWriter         文件写入器
    FolderWriter       文件夹写入器
    AreaManager        元数据域管理器

helper      辅助工具   便于辅助开发及代码复用
    ArrReader     字符串数组读取器
    Indent        用于管理字符串的缩进
    Log           记录并管理日志

migrate     数据迁移相关
    SqlServer            建立数据库连接及进行查询
    MigrateData          迁移数据类
    ConversionMethod     数据转换类

database    数据库操作相关
    DbConnector          选择并返回与数据库实例的连接
    DbEntity             数据库实体，可执行查询操作，并附带常用指令集
    
interface   系统界面   用户交互相关
    BaseInterface      命令行界面父类
    MainInterface      命令行主界面
    WorkSpaceInterface 工作空间界面
